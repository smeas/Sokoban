# Sokoban
Sokoban written in C# with SFML.

Some levels were taken from older sokoban games.

### Controls
|             |                   |
| ----------- | ----------------- |
| Movement    | wasd / arrow keys |
| Reset level | r                 |
| Next level  | space             |
